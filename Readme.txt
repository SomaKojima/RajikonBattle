制作者：小嶋颯天
タイトル：ラジバト

----始め方------------------------------------------------
実行ファイル内の「らじこんバトル.exe」を実行する

----フォルダの説明------------------------------------------------
「ドキュメント」　…　企画書・仕様書などが入っています
「プログラム」　　…　ソリューションやcppなど制作したプログラミングが入っています
「実行」　　　　　…　exeと実行に必要なファイルが入っています
「動画」　　　　　…　プレイ動画が入っています


----操作方法------------------------------------------------
終わり方
Escapeキーを押す

車の操作
W　…　前に進む
A　…　左に進む
S　…　後ろ進む
D　…　右に進む

マウスの移動　…　カメラの視点を移動させる
左クリック　…　弾を撃つ


----開発環境------------------------------------------------
DirectXTK
